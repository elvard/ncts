ncts
====

curses/python3 frontend for [Task Spooler](http://vicerveza.homeunix.net/~viric/soft/ts/)

**Status:** Works, but looks ugly. Consider it unstable.

commands
--------

-  ``arrow keys`` Move
-  ``Esc`` Unselect current task (will show last one by default)
-  ``d`` Delete task
-  ``q``, ``Q`` Exit app
